# Middleware

Composable middleware layers for Tower-compatible services. Middleware follows the Tower service pattern for type-safe, zero-cost request/response transformation, side effects, error handling, and behavior modification.

## Architecture Overview

```
┌─────────────────────────────────────────┐
│ Middleware Layer 1                     │  ← Outermost middleware
├─────────────────────────────────────────┤
│ Middleware Layer 2                     │  ← Nested middleware
├─────────────────────────────────────────┤
│ Service                                 │  ← Core service
├─────────────────────────────────────────┤
│ Network                                 │  ← Network communication
└─────────────────────────────────────────┘
```

**Request Flow:** Client → Middleware 1 → Middleware 2 → Service → Network

**Response Flow:** Network → Service → Middleware 2 → Middleware 1 → Client
