## Architecture (at a glance)

```
┌─────────────────────────────────────────┐
│ Client (chains, apps)                   │  ← High-level client APIs
├─────────────────────────────────────────┤
│ Protocol Services (JSON-RPC, gRPC)      │  ← Protocol-specific services
├─────────────────────────────────────────┤
│ Middleware (service-level)              │  ← Service-level middleware
├─────────────────────────────────────────┤
│ Transports (HTTP / WebSocket)           │  ← Transport layer
├─────────────────────────────────────────┤
│ Middleware (transport-level)            │  ← Transport-level middleware
├─────────────────────────────────────────┤
│ Network                                 │  ← Network communication
└─────────────────────────────────────────┘
```
